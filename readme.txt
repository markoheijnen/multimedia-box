== Description ==



== Installation ==

1. Upload the folder `multimedia-box` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the API code in your own plugin/theme to enable a multimedia box for a specific post type
